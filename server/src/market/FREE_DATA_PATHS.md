# 무료 시장 데이터 경로 검증 (Free-data path verification)

> SPEC 피드백 라운드 3 §5.6 open question 대응. "KR 무료·합법 경로는 착수 전 실증
> 필요 — 단정 ✗". 부재/접근불가를 검증 없이 단정하지 않고, 실제로 확인된 경로와
> 미검증/대체 처리를 아래에 박제한다. 코드: `server/src/market/overview.ts`.

## 상세

### 1. 지수 (✅ 검증됨)
- **경로**: Yahoo Finance v8 chart (`query1.finance.yahoo.com/v8/finance/chart/<sym>`),
  무료·keyless. 코드베이스의 US/KR 어댑터가 동일 경로를 이미 사용·테스트(`us.test.ts`,
  `kr.test.ts`). 지수 심볼(`^KS11` 코스피, `^KQ11` 코스닥, `^IXIC` 나스닥, `^GSPC` S&P)도
  동일하게 resolve.
- **전일대비·등락률**: 최근 2개 일봉 종가 차이로 결정적 계산(`quoteFromCandles`).
- **granularity**: 일봉(EOD), `delayed: true` 정직 표기. 실시간 틱 아님(§5.7).

### 2. TOP 상승/하락/거래상위 (✅ 무료 자체 산출 — 유니버스 범위)
- **왜 자체 산출인가**: keyless·무료·합법으로 *거래소 전체* 순위를 반환하는 안정 API가
  없다(KRX 데이터 서비스는 폼/키 게이트, 포털 스크래핑은 ToS 리스크). 따라서 부재를
  단정하지도, 데이터를 날조하지도 않고 — 이미 수집하는 **추적 유니버스 일봉**에서
  등락률(상승/하락)·거래량(거래상위)을 결정적으로 계산한다.
- **정직성 플래그**: 각 행 `scope: "universe"` + overview `notes`에 "거래소 전체 순위
  아님" 명시. 거래소 전역 순위로 오인되지 않게 한다.
- 향후 무료·합법 전역 순위 경로가 실증되면 어댑터 교체만으로 승급(다운스트림 무변경).

### 3. 시가총액 TOP (⚠ 생략)
- 시총 = 주가 × 발행주식수. 발행주식수는 무료 일봉에 없다. 무료 keyless 소스 실증 전까지
  **생략**한다(추정 날조 ✗ — `feedback_no_unverified_negative_claims` 정합).

### 4. 인기 검색 종목 (⚠ 프록시 — 후보 풀 합류)
- 무료·keyless 검색량/인기 소스 미확인. **\|등락률\| 기반 주목도 프록시**로 대체하고
  `notes`에 프록시임을 명시. SPEC §5.2-3대로 추천 배치 후보 풀에 합류(`popularSymbols`).
- 소스 방식과 무관하게 후보 풀 보강 역할은 유지된다.

### 5. 업종·테마 상위 (내부 전용)
- SPEC §5.2 비채택: UI 전면 노출 ✗, 추천 엔진 *내부 입력*으로만 보관. 무료 섹터 매핑이
  아직 번들되지 않아 `SectorThemeLeader` 슬롯은 비어 있다(섹터 날조 ✗). 공개 overview에
  직렬화되지 않는다.

### 6. 거시 패널 (⊘ 비채택)
- 환율·금리·유가·금·원자재: §5.7 비채택. 수집하지 않는다.
