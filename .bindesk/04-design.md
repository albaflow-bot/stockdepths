## 디자인 구성기 — 페이지 구조 지시 (W43-DC)
> ⚡구성기로 확정된 결정론적 지시. 코드 생성 시 이 구조·스타일을 그대로 반영할 것.

- **레이아웃**: 1단
- **메뉴 위치**: 상단
- **콘텐츠 구조**: 카드 그리드
- **스타일 기법**: 미니멀(여백·절제·또렷한 위계)
- **인터랙션 효과**:
  - 마이크로 인터랙션(작은 피드백 애니메이션)
- **색 팔레트 / 명도**: mono / 라이트
- **기준 페이지 종류**: 랜딩 (히어로 + 섹션 스크롤, 메뉴는 상단 고정 권장)

### 페이지 종류별 융통성
같은 디자인 언어(색·타이포·radius)는 유지하되, 레이아웃은 페이지 성격에 맞게 조정:
- 랜딩: 히어로 중심 + 섹션 스크롤(좌/우 메뉴 대신 상단 고정 권장)
- 게시판/피드: 위 단 수·구조 반영, 목록 가독성 우선
- 로그인: 단 수 무시, 중앙 단일 폼 카드
- 상세: 1단 본문 가독성 우선 + 보조 사이드
